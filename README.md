# 🏡 Digital Village Web Application

Aplikasi **Digital Village** merupakan sistem berbasis **web native** yang dirancang untuk membantu digitalisasi informasi dan layanan desa agar dapat diakses dengan mudah oleh masyarakat.

Website ini menyediakan berbagai informasi penting mengenai desa seperti **profil desa, berita desa, informasi kegiatan, serta layanan administrasi dasar** yang dapat membantu masyarakat memperoleh informasi dengan lebih cepat dan efisien.

Sistem ini bertujuan untuk mendukung proses **transformasi digital desa**, sehingga pelayanan kepada masyarakat menjadi lebih transparan, informatif, dan mudah diakses melalui platform online.

Project ini dikembangkan sebagai bagian dari **pembelajaran pengembangan web native pada semester 3** serta sebagai latihan dalam membangun sistem informasi berbasis web menggunakan teknologi dasar web development.

---

# 🚀 Features

Beberapa fitur utama dalam aplikasi ini antara lain:

### 🏡 Profil Desa
Menampilkan informasi lengkap mengenai desa seperti sejarah desa, visi misi, serta struktur organisasi pemerintahan desa.

### 📰 Berita dan Informasi Desa
Menyediakan informasi terbaru mengenai kegiatan desa, pengumuman, dan berita penting yang dapat diakses oleh masyarakat.

### 🖼️ Galeri Kegiatan
Menampilkan dokumentasi kegiatan desa dalam bentuk foto atau media lainnya.

### 📢 Pengumuman Desa
Menyediakan informasi pengumuman resmi dari pemerintah desa kepada masyarakat.

### 📱 Responsive Web Design
Website dapat diakses dengan baik melalui berbagai perangkat seperti desktop, tablet, maupun smartphone.

# ⚙️ Installation

Ikuti langkah berikut untuk menjalankan project secara lokal.

### 1. Clone repository

```
git clone https://github.com/username/digital-village.git
```

### 2. Masuk ke folder project

```
cd digital-village
```

### 3. Pindahkan project ke folder web server

Letakkan folder project ke dalam folder server seperti:

```
htdocs (XAMPP)
atau
www (Laragon)
```

### 4. Setup database

1. Buka **phpMyAdmin**
2. Buat database baru
3. Import file database yang tersedia pada folder:

```
database/
```

### 5. Jalankan project

Buka browser dan akses:

```
http://localhost/digital-village
```
